GraphQL

GraphQL is an API query language, and a server-side runtime for executing queries using a custom-defined type system. GraphQL isn't tied to any specific database or storage engine.

For more information, head over to the official documentation website and their extensive documentation, in this section we only cover the basics to work with the Squid GraphQL Gateway, with short explanations of how to perform GraphQL queries, how to paginate and sort results.
