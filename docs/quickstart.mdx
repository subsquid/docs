# Quickstart


## Pre-requisites

- node 16.x
- docker
- npm (the template is not compatible with `yarn`)
- [make](https://www.gnu.org/software/make/)

On Windows we recommend using [WSL](https://docs.microsoft.com/en-us/windows/wsl/).

For more information on how to set up the Development Environment, take a look at the [Tutorial on this topic](/docs/tutorials/development-environment-set-up)


## Clone <span><a href="https://github.com/subsquid/squid-template">squid-template</a></span>

```bash
git clone https://github.com/subsquid/squid-template.git 
cd squid-template
```

##  Install dependencies

```bash
npm ci
```

## Build the squid

```bash
make build
```
