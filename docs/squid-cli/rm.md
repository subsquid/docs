## `sqd rm NAMEANDVERSION`

Remove a squid or a squid version

```
USAGE
  $ sqd rm NAMEANDVERSION

ARGUMENTS
  NAMEANDVERSION  <name> or <name@version>

ALIASES
  $ sqd squid:kill
  $ sqd kill
```
