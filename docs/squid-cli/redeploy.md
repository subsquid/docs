## `sqd redeploy NAMEANDVERSION`

Restart a squid version

```
USAGE
  $ sqd redeploy NAMEANDVERSION

ARGUMENTS
  NAMEANDVERSION  name@version

OPTIONS
  -e, --env=env      environment variable
  --envFile=envFile  file with environment variables

ALIASES
  $ sqd squid:redeploy
```
