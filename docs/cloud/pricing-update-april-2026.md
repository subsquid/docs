# Pricing Update: April 2026

:::info Effective Date
**Effective Date:** April 1, 2026

Cloud hosting prices will increase by 20% and RPC add-on pricing will update to $5 per 1M requests.
:::

## What's changing?

We're updating our pricing structure effective **April 1, 2026**. Here's a summary of the changes:

* **Cloud Hosting:** 20% increase on all compute resources (API, Processor, Database, Collocated Squids)
* **Storage:** 20% increase (`$0.50` → `$0.60` per GB/month)
* **RPC Add-on:** Overage pricing increases from `$2.00` to `$5.00` per 1M requests
* **Free Tier:** Unchanged - 2M RPC requests per month remain free

### API & Processor Compute

| Profile | Current Price/hr (until March 31) | New Price/hr (April 1) |
| ------- | --------------------------------- | ---------------------- |
| small   | \$0.04                            | \$0.048                |
| medium  | \$0.08                            | \$0.096                |
| large   | \$0.15                            | \$0.18                 |
| xlarge  | \$0.30                            | \$0.36                 |
| 2xlarge | \$0.60                            | \$0.72                 |

### Database Compute

| Profile | Current Price/hr (until March 31) | New Price/hr (April 1) |
| ------- | --------------------------------- | ---------------------- |
| small   | \$0.08                            | \$0.096                |
| medium  | \$0.16                            | \$0.192                |
| large   | \$0.33                            | \$0.396                |
| xlarge  | \$0.66                            | \$0.792                |
| 2xlarge | \$1.32                            | \$1.584                |

### Collocated Squids

* **Current:** $0.02/hr (~$15/month)
* **New (April 1):** $0.024/hr (~$17/month)

### Storage

* **Current:** \$0.50/GB/month
* **New (April 1):** \$0.60/GB/month

### RPC Requests

* **Free Tier:** 2M requests/month (unchanged)
* **Overage - Current:** \$2.00 per 1M requests
* **Overage - New (April 1):** \$5.00 per 1M requests

## When does this take effect?

The new pricing will be effective **April 1, 2026**. All usage from that date forward will be billed at the new rates.

* **Until March 31, 2026:** Current pricing remains in effect
* **From April 1, 2026:** New pricing applies to all usage

:::tip
You can review your current usage and estimate the impact of these changes in your [Cloud Dashboard billing section](https://app.subsquid.io/billing/invoices).
:::
